🚀 HR Resume Screening AI

An AI-powered system that automates resume screening, candidate matching, and interview question generation using a hybrid approach of NLP + Large Language Models (LLMs).

📌 Features
📄 Resume Parsing (PDF/DOCX)
🧠 Skill Extraction & Analysis
🎯 Job Matching with scoring
💬 AI-based Interview Question Generation
📊 Analytics Dashboard
🤖 LLM Integration

Uses OpenAI GPT models
Enables context-aware parsing & matching
Includes fallback to traditional NLP

🧮 Matching Formula
Score = (Skills × 0.5) + (Experience × 0.3) + (Education × 0.2)
